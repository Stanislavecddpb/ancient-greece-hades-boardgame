# Cyclades (браузерная версия)

Цифровая реализация настольной игры про борьбу за господство над островами
Эгейского моря. Сделана, чтобы играть с друзьями по сети.

## Цель игры

Первым собрать **2 Метрополии**. Метрополию можно получить тремя способами:

- построить все 4 типа зданий (Порт, Крепость, Храм, Университет);
- собрать 4 Философов;
- захватить чужую Метрополию в бою.

Ключевая механика — **скрытый аукцион** за расположение богов. Каждый цикл боги
(Арес, Посейдон, Зевс, Афина и Аполлон) дают разные действия, и игроки тайно
торгуются золотом за право выбрать нужного бога первым.

### Доход

Рога изобилия лежат на клетках доски. Рог **на воде** даёт золото тому, чей флот
стоит на этой клетке; рог **на суше** — владельцу острова. На доске на 4 игроков —
6 рогов на воде по краям плюс рога на суше (на части островов по два).

## Структура проекта

Монорепозиторий (npm workspaces) на базе **boardgame.io**:

- `packages/engine` — чистая игровая логика на TypeScript (правила, доска, бой).
- `packages/server` — авторитарный сервер на boardgame.io: комнаты, синхронизация.
- `packages/client` — клиент на React + Vite: SVG-доска (вид сверху), интерфейс.

## Как запустить

Сначала один раз поставить зависимости:

```bash
npm install
```

### Вариант 1. Играть (как в проде) — один порт

Сервер и клиент работают на **одном порту 3001**: сервер раздаёт собранный
клиент и держит сокеты.

```bash
npm run play      # собрать клиент и запустить сервер на http://localhost:3001
```

Если клиент уже собран и менять код не нужно — быстрее:

```bash
npm run serve     # просто поднять сервер на http://localhost:3001
```

Открыть `http://localhost:3001`:
- **Создать игру** → выбрать число игроков → попасть в **лобби**.
- В лобби ждём, пока все займут места; затем **хост** жмёт «Начать игру».
- При старте проигрывается заставка и определяется очерёдность хода.
- **Локальная игра** (`#/local`) — хотсит за одним компьютером (старт сразу).

### Вариант 2. Разработка — с горячей перезагрузкой

Два процесса: сервер на :3001 и Vite-клиент на :5173 (HMR).

```bash
npm run dev:server   # сервер на :3001
npm run dev:client   # клиент на :5173 (открыть этот адрес)
```

### Играть с друзьями по интернету

Деплой не нужен — поднимаешь у себя и отдаёшь ссылку через туннель (ngrok).
Пошагово, с постоянным адресом — см. **[NETPLAY.md](NETPLAY.md)**.

> Важно: всегда запускай свежий сервер из этого кода. Если на :3001 «висит»
> старый процесс, перезапусти его (`taskkill /F /IM node.exe`, затем `npm run serve`).

## Статус и дорожная карта

Цель — **доведённая до конца игра**, а не прототип. Базовая игра целиком играбельна.

**Готово:**
- Полный цикл: лобби со стартом хоста → доход → аукцион богов (с вытеснением
  ставок и скидкой за жрецов) → действия → проверка победы.
- Очерёдность хода: первый игрок определяется случайно, заставка-занавес и
  анимация очерёдности на старте; в фазе действий боги ходят сверху вниз.
- Действия богов: наём войск/флота/жрецов/философов, постройка зданий.
- Движение флота по правилам (1 монета, до 3 клеток, высадка по пути) и войск
  по «мосту» из кораблей — стрелками прямо на карте.
- **Интерактивный бой** с отступлением, захват островов/Метрополий, бонусы
  крепости/порта; кости с гранями 0,0,1,1,2,3.
- Мифические существа: колода и рынок из 3, эффекты, покупка за действие
  (Храм снижает цену), бесплатная прокрутка Зевсом.
- Аполлон: доход в зависимости от числа островов + установка рога изобилия.
- Метрополии (4 здания / 4 философа / захват), условие победы — 2 Метрополии.
- Доска на 4 игроков по точным координатам, рога изобилия и доход с них.
- Сетевая игра по ссылке (лобби, комнаты, переподключение) на одном порту;
  имена игроков, туман войны; локальный хотсит.
- Визуал: круглое поле с рельефными островами, фишки-воины/триремы, фото-карты
  богов, поле богов со ставочными дорожками.

**Дальше:**
1. **Туман войны на сервере** (`playerView`) — сейчас скрытие на клиенте.
2. **Карты на 2, 3 и 5 игроков** (сейчас точная — только на 4).
3. **Постоянный деплой**, чтобы не держать свой ПК включённым.
4. Полировка: анимации боя, звук, мобильная раскладка.

Вне ближайших планов: дополнения (Hades, Titans, Monuments).
