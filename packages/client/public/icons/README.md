# Иконки на карте

Положи сюда PNG (лучше с прозрачным фоном — лягут на карту). Имена ровно такие:

| Что | Файл |
|-----|------|
| Крепость | `fortress.png` |
| Порт | `port.png` |
| Храм | `temple.png` |
| Университет | `university.png` |
| Метрополия | `metropolis.png` |
| Монета | `coin.png` |
| Рог изобилия | `cornucopia.png` |

Если файла нет — остаётся прежний рисованный значок (откат), приложение не падает.

После добавления файлов нужно **пересобрать** клиент (`npm run play`), потому что
`public/` копируется в `dist/` при сборке. В режиме `npm run dev:client` —
подхватывается сразу.
