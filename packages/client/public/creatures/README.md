# Карточки мифических существ (арт)

Файлы кладём сюда, расширение `.jpg`. Если файла нет — показывается
запасной вид (эмблема/подпись), приложение не падает.

## Рубашка и сброс (горизонтальные, ~3:2)

| Что | Файл |
|-----|------|
| Рубашка колоды | `cardback.jpg` |
| Сброс | `discard.jpg` |

## Лицевые карты существ (портрет, ~2:3)

Имя файла = id существа из движка. Текущий набор:

`minotaur.jpg`, `pegasus.jpg`, `cyclops.jpg`, `kraken.jpg`,
`harpies.jpg`, `satyrs.jpg`, `nymph.jpg`, `muse.jpg`

> Набор существ в движке пока свой и может не совпадать с физическими
> картами. Если хочешь точный набор как в коробке — пришли список существ
> (название, эффект, цена), приведём движок к нему и тогда добавим арт.

После добавления файлов сервер перезапускать не нужно — обнови страницу.
