# Карточки богов (арт)

Сюда нужно положить 5 фотографий карточек богов. Имена файлов строго такие
(всё латиницей, расширение `.jpg`):

| Бог       | Файл           |
|-----------|----------------|
| Арес      | `ares.jpg`     |
| Посейдон  | `poseidon.jpg` |
| Зевс      | `zeus.jpg`     |
| Афина     | `athena.jpg`   |
| Аполлон   | `apollo.jpg`   |

После того как файлы окажутся здесь, перезапускать сервер не нужно — обнови
страницу, и карточки подхватятся автоматически.

Если файла нет, на его месте показывается прежнее текстовое описание
(приложение не падает).

Совет: обрежьте фото по рамке карточки (без фона стола) и по возможности
выровняйте — панель узкая, всё лишнее по краям будет незаметным.
